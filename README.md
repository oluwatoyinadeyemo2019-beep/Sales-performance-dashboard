# Sales Performance Dashboard

## Project Overview
This project analyzes sales performance across months, cities, products, and order hours.

## Dashboard Insights
- Total Revenue: 33.6M
- Total Orders: 41M
- Peak sales occur in later months of the year
- Some cities generate significantly higher revenue than others
- Laptop products dominate the top revenue positions

## Dashboard Features
- Revenue by Month trend analysis
- Revenue by City comparison
- Orders by Hour distribution
- Top 5 products by revenue

## Tools Used
- Excel / Power BI
- Data Visualization
- Business Intelligence
